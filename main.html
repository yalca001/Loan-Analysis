Loan Prediction Data Analysis


Welcome to our project!

The aim is to develop an end to end web application that helps a user to determine his chances of getting a loan. This is done by implementing several different statistical learning techniques to try to estimate the probability of default for each loan. Machine learning models are used to classify a set of unseen data and statistical metrics are used to compare the results. The finalized model is then deployed using a flask library on the Heroku servers and a website is created for the user to navigate and predict his chances of acquiring a loan. In order to analyze our data, we created a RESTful API using the CSV file with Flask. Then we used HTML, Javascript, Plotly and D3 to come up with cool maps and graphics.

To map all the coordinates, we made a map with all the pick-up locations represented by markers. Since there were so many locations, we created a cluster map to make it quick and easy to spot how many pick-ups were made at a given area. Finally, we created a heatmap to show (quite literally) pick up hot spots througout New York City.

In terms of data analysis, we decided to check the frequency of pick-ups by the hour and every three hours to see what times people are using Lyft more frequently.
